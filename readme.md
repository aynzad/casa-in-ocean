## Three.js Donuts

### A simple three.js project to create a casa in ocean

### [Live Demo](https://casa-in-ocean.vercel.app)

## How to run

### Clone the repository

```bash
git clone https://github.com/aynzad/casa-in-ocean
```

### Install dependencies

```bash
yarn
```

### Run the project

```bash
yarn dev
```
